# Scrabble Course — public lessons

The public, web-hosted lessons for a self-paced course on competitive Scrabble
(North American **NWL2023** word list).

**Live site:** https://jvisick.github.io/scrabble-course/

This repo holds only the student-facing course content — lessons, reference sheets,
and shared assets. Personal progress tracking (missions, learning records, notes)
lives in a separate private repo and is intentionally *not* published here.

## Contents

| Path | What's in it |
|------|--------------|
| `index.html` | Landing page linking every lesson and reference sheet |
| `lessons/` | Self-contained HTML lessons, one concept each (start: `0001-*.html`) |
| `reference/` | Printable cheat-sheets (the 107 two-letter words, the skill roadmap) |
| `assets/` | Shared stylesheet + interactive quiz/flashcard widgets |

Everything is static HTML/CSS/JS — no build step. Open `index.html` locally, or
visit the live site above.
